Overview

What is DermaFusion?

DermaFusion is a multimodal, explainable AI diagnostic support system developed as a Software Engineering research project. It addresses three critical gaps in existing AI dermatology tools — the black-box problem, context blindness, and demographic bias — by fusing visual deep learning with structured clinical metadata and Grad-CAM explainability.

Architecture

Three-pipeline design

01
Visual Stream
MobileNetV2 CNN
Extracts 1280-dimensional feature vectors from dermoscopic images. Preprocessing includes black-hat morphological hair removal and channel-standardised colour normalisation to reduce device and skin-tone bias.
02
Clinical Stream
Multi-Layer Perceptron
Encodes patient age (normalised), biological sex (one-hot, 3 dims), and anatomical site (one-hot, 15 dims) into a 32-dimensional clinical feature vector — the context a clinician naturally considers.
03
Fusion + XAI
Concatenation + Grad-CAM
Both feature vectors are concatenated (1312-dim) and passed through fully-connected layers. Grad-CAM backpropagates the prediction to the final convolutional layer, generating a pixel-level attention heatmap.

Technology

Built with

🔥
PyTorch + TorchVision
Deep learning framework. MobileNetV2 backbone from torchvision.models with custom fusion head.
🗺
Grad-CAM (Custom)
Built-in gradient-based class activation mapping using PyTorch hooks. No external library required.
👁
OpenCV
Hair removal via black-hat morphological transform and inpainting. Colour normalisation for skin-tone equity.
🌐
Flask
Lightweight Python web server. REST API endpoint for inference. Serves HTML/CSS/JS frontend.
📊
HAM10000 Dataset
10,015 dermoscopic images across 7 classes. ISIC Archive. Train/Val/Test split 75/15/10.
📐
scikit-learn
Classification report — precision, recall, F1-score per class. Used for test evaluation only.
